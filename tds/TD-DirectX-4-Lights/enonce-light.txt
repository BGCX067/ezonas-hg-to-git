DirectX Light
Le but de ce TD est de vous familiariser avec le fonctionnement des lights en directX. Pour cela vous 
mplémenterez 3 classes de lumière afin d'éclairer le cube que vous avez réalisé dans le précédent TD,
Instructions : 
1°) A partir de la classe Light qui vous est fourni, héritez les classes Directional, Point, puis Spot
2°) Compléter la classe Light avec les fonctions suivantes : 
• SetIndice
• SetDiffuse
• SetSpecular
• SetAmbiant
• SetOn
• SetOff
• IsLightOn
Ainsi que les accesseurs nécessaires pour les classes filles
• SetDirection
• SetPosition
• SetAttenuation
• SetRange
• SetCone
Eclairez votre Cube de voyage : un spot vert a gauche, un spot bleu à droite, une pointlight rouge au dessus.
Et une directional light switchable en rouge, vert bleu blanc.
Vous pouvez également déplacer les lumières.
Questions subsidiaires :
3°) Complétez le LightManager
